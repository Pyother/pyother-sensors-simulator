# pyother-sensors-simulator

## Installation

1. Ensure you have **Node.js** and **npm** installed.  
2. Install the package globally:  
   ```bash
   npm i -g pyother-sensors-simulator
   ```
3. Run the simulator:
   ```bash
   pyother-sensors-simulator
   ```
4. The application will be available at: http://127.0.0.1:5000
